# USCC Chat UI v3

本版本将首页改为 ChatGPT 式单对话框界面，并做了以下调整：

- 上传按钮移动到输入框左侧。
- 左侧“病例知识库”可点击进入 `/knowledge`。
- 知识库页支持搜索、分组筛选、点击病例详情、手动添加病例。
- 首页左侧也支持“添加当前病例”。
- 在对话中输入“投喂进入数据库 / 加入数据库 / 添加病例 / 存入知识库”会将当前对话整理为新增病例。
- 默认仅显示 3 组最相似病例。
- 取消前端风险评分、风险提醒、主要依据卡片显示。
- 对话记忆以单个聊天为单位；新建对话不会自动继承其他对话的病例上下文。
- 上传 jpg/png/webp/tif 等图片后会作为当前对话附件保存。若配置 `OPENAI_API_KEY`，后续问答会把图片作为多模态输入传给模型；未配置时只保存附件。

运行方式仍然是：

```powershell
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
python app.py
```

打开：

```text
http://127.0.0.1:5000
```
