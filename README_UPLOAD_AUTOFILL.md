# 上传与自动回填说明

## 新增功能

1. 影像/病理附件上传：支持 `.jpg`、`.jpeg`、`.png`、`.webp`、`.bmp`、`.tif`、`.tiff`、`.dcm`、`.pdf`。
2. 病例文件自动回填：支持 `.xlsx`、`.xls`、`.csv`、`.docx`、`.txt`。
3. 上传病例 Excel/Word 后，系统会尝试识别并回填：性别、年龄、LS 状态、TNM/分期、Grade/分化程度、淋巴结状态、病史、既往尿道操作史、症状体征、肿瘤局部描述、影像摘要、病理摘要和自由文本。

## 注意

- 旧版 Word `.doc` 不是 XML 格式，当前不解析。请用 Word 打开后另存为 `.docx` 再上传。
- JPG/PNG 等图片当前只做附件保存，不做自动影像诊断。请将影像报告文字粘贴到“影像/尿道镜摘要”。
- 自动回填是规则抽取，上传后必须人工核对，尤其是年龄、LS 状态、TNM 和病理信息。

## 本地免 PowerShell 可见窗口运行

双击：

```text
start_windows_background.vbs
```

它会在后台启动服务并自动打开浏览器。缺点是：你的电脑关机、休眠或脚本进程停止后，别人就无法访问。

## 公开网址部署

如果想让任何人只打开网址即可访问，请把项目上传 GitHub 并部署到 Render/Railway 等云平台。部署成功后，服务器会替你长期运行 Flask，不需要你本机开着 PowerShell。
