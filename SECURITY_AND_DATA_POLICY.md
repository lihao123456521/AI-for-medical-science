# Security and Data Policy

## Public repository rule

This repository is for software code and synthetic demo data only.

Do not commit:

- real patient names, identifiers, dates, records, images, or protected health information;
- original research spreadsheets or exported hospital data;
- runtime uploads or locally added cases;
- `.env` files, API keys, tokens, passwords, or private URLs.

## Synthetic demo data

`data/knowledge_base.xlsx` contains generated `DEMO-*` records. They are fictional combinations created for interface testing and do not correspond to real patients.

## Research use

If the software is used with real research data, keep that data in a controlled environment with appropriate authorization, ethics review, access control, retention rules, and audit practices.

## Medical disclaimer

This prototype does not provide diagnosis, staging, or treatment recommendations. Outputs require professional review.
