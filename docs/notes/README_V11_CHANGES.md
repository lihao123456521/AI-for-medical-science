# v11 更新说明

1. 回答时会检索已投喂文章，并在回答卡片下方显示“可参考文献”。
   - 文章匹配不再只依赖整句精确匹配，增加了 SCC、LS、尿道、狭窄、手术、治疗、复发、病理等医学关键词扩展。
   - 文献卡片会显示命中词、可借鉴片段和文章详情链接。

2. 添加病例页支持一键投喂病例文件。
   - 单文件解析并自动填充：PDF、DOCX、Excel、CSV、TXT、MD。
   - 文件夹批量投喂：选择病例文件夹后一键解析并加入病例知识库。

3. 病例知识库新增“一键删除当前结果”。
   - 删除前会弹出提醒。
   - 需要手动输入 DELETE 二次确认。
   - 删除后病例会记录到 data/deleted_cases.json，不再参与检索。
