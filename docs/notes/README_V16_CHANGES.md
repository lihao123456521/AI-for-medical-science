# v16 更新说明

1. 修复 Excel 双层表头识别错误：`基本信息/临床症状/治疗方案` 等分组标题与 `姓名/性别/年龄/病理诊断` 等字段提示不会再被识别为单独患者。
2. 对话中候选患者确认改为“勾选式”交互：点击候选病例卡片中的“选择该患者”，系统会在当前对话内打勾并把该患者信息合并到当前会话上下文，不再跳转离开对话页。
3. 字体设置改为弹窗：左侧点击“Aa 字体设置”打开配置窗口；字体大小改为数字选项（13、14、15、16、17、18、20、22）。
4. 前端静态资源版本号更新为 `chatui16`，更新后请按 `Ctrl + F5` 强制刷新。

数据仍默认保存在用户目录：

```text
C:\Users\你的用户名\.uscc_scc_flask_data
```

更新前端时不要删除该目录。
