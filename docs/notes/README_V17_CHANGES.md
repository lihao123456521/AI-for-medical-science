# v17 更新说明

1. 左侧按钮「Aa 字体设置」改名为「字体设置」。
2. 相似病例模块只列出相似病例与治疗/转归要点，不在该模块里做综合建议。
3. 手术/用药讨论方向改为分行输出：
   - 病例 XXX：分析……建议……
   - 文献《XXX》：分析……建议……
4. 相似病例和参考文献不再强制输出 4 个；后端会根据匹配强度筛掉弱相关条目，最多各 4 个。
5. 添加病例页按钮重排：上传医学影像在上方；其下为「上传病例文件 + 解析文件」；再下方为「一键选择病例 + 一键解析」；底部提供返回知识库。
6. 单个病例文件解析后会自动保存；批量病例文件仍会弹出候选病例窗口，由医生确认后加入知识库。
