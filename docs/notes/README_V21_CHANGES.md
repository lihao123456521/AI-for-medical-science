# V21 更新说明

1. 对话框上传病例表后，会立即在当前对话中显示候选患者卡片；医生选择患者后，系统自动在当前对话内生成初步分析，不跳转页面。
2. 添加病例页移除了“一键选择病例”和“一键解析”，保留“上传病例文件 + 解析文件”。病例文件支持 PDF、DOCX、Excel、CSV、TXT、MD，可一次选择多个文件。
3. 修复单个 Excel 内识别到多条病例时前端只显示少数候选的问题；后端会返回完整候选列表（上限 2000 条），医生可按需勾选录入。
4. 文章投喂页移除了“一键选择文章”和“一键解析”，改为“选择一个或多个文章文件 + 解析并保存文件”，并新增“保存手动输入文章”。
5. 选择候选患者后会自动调用 `/api/chat`，如配置 API Key 会调用所选模型；未配置时使用本地知识库兜底。
